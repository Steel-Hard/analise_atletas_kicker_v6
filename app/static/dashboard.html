Distribuição de Classificação das Sessões

Resultado de classificar_sessao(): combina o Isolation Forest (gatilho de anomalia) com o Z-Score Composto (direção). Sessões só entram em "Alta" ou "Queda" quando o IF marcou if_anomalia = True e o Z-Score Composto passa do limiar (±1.5).

Anomalias por Atleta (Isolation Forest)

Quantidade de sessões marcadas como anômalas (if_label = -1) por atleta. Serve para checar se o modelo treinado por atleta está sinalizando uma proporção plausível de anomalias (esperado ≈ contamination).

Comparação de Perfis (Radar)

Perfil = média dos valores brutos de cada atleta, padronizada globalmente (entre atletas). Cada eixo mostra quantos desvios-padrão o atleta está acima/abaixo da média do elenco naquela métrica. Se este radar estiver "achatado" (tudo perto de 0 para todos), o cálculo de perfil voltou a ficar errado — selecione 2 a 4 atletas para comparar.

Histórico do IF Score (Ponto)

Score do Isolation Forest ao longo do tempo para o atleta selecionado. Quanto mais negativo, mais anômala a sessão. Pontos em âmbar foram marcados como anomalia (if_anomalia = True).

Sugestão de Substitutos (Similaridade)

Similaridade combinada (40% distância euclidiana + 60% distância de cosseno) entre o perfil do atleta de referência e os demais. Os valores devem variar entre candidatos — se todos derem exatamente o mesmo número, o cálculo de perfil está degenerado.

Atletas

Visão geral por atleta: total de sessões, anomalias detectadas e status da última sessão classificada.
